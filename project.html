Project idea:

A Lamp that activates with motion to enable moving around room less dangerous. There are 3 color modes avaliable for the user to choose:

There are 3 buttons to Change Color , Change brightness and Manually off the light.

I used THINKERCADD to plan and test out the coding for arduino and the wiring before doing it physically. This step is very helpful and crucial as it ironed out many of the mistakes i had and allowed me to test the software part without any physical component issues.
